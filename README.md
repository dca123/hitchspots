# HitchSpots

Remake of the now broken hitchwiki.com/maps as a mobile app. Designed with Figma and developed with Flutter. 

## Features
- Add hitching spots
- Review and rate hitching spots
- View nearby hitchiking spots by rating
    - Red => Bad
    - Orange => Medium
    - Green => Good
